■ 基本情報
- アプリ名: タスク管理アプリ（task-app）
- 利用者: 個人ユーザー（ログインして利用）
- 目的: やるべきタスクを管理し、進捗を把握する

■ タスクについて
- タイトル: 必須
- 説明: 任意（長文OK）
- 優先度: 高・中・低の3段階
- カテゴリー: システム共通（仕事、プライベート、勉強など）

■ カテゴリーについて
- システム共通のカテゴリーを用意
- ユーザーが自由に追加・編集・削除できる
- タスクは必ず1つのカテゴリーに属する

■ ユーザーについて
- ユーザー登録・ログイン・ログアウトが必要
- 自分のタスクのみ表示・編集・削除できる
- 他のユーザーのタスクは見えない

■ API機能
- 認証なしでタスク一覧・詳細を取得できる公開API
- 外部サービスとの連携を想定

■ 今回のスコープ外
- タスクの期限（due_date）
- タスクの完了ステータス
- 通知機能
- チーム共有機能
